== Description ==

This plugin provides shortcode for add ticket fields in contact form 7.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `ticket-booking` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
